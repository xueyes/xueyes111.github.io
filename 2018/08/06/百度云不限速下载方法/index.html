关于不限速下载的方法推荐几个实用的：

1、百度网盘下载助手脚本 —

最早是网友“有一份田”制作的脚本百度下载助手、可以显示下载直链、绕过大文件云盘下载、还可以调用迅雷或者【IDM】加速下载。

顺便推荐下【Internet Download Manager】这是我用过下载速度最快的下载器、随便甩迅雷几条街、附上破解版下载链接

使用方法：

  1. 浏览器安装拓展： violentmonkey 或者： Tampermonkey
  2. 安装下载脚本：greasyfork
  3. 用迅雷或IDM下载时，点击“直接下载”，然后将链接复制到迅雷或IDM中去下载，支持在网盘管理页面和文件分享页面生效。

注意：

  1. IDM绿色版需要点击运行文件 “!绿化卸载.bat” 才会自动关联下载。
  2. 如果你不太会怎么操作，你可以尝试一下已经集成好脚本的Firefox浏览器：https://www.runningcheese.com/v10（下载后需要在右上角的那个“火箭”图标里选择“系统工具”，然后再选择“设置IDM”，这样就会自动调用IDM下载了。）
  3. 如果下载速度被限制在100KB左右，说明你的帐号被百度网盘拉黑了，需要退出登录才能恢复正常下载。
  4. 如果下载速度在200-700KB左右，可以在“设置–>连接”中，将最达连接数设置为16或者32.
  5. 在文件分享页面，直接选择文件夹会无法下载，可以同时勾选下载任意一个文件，让文件夹与文件同时下载，这时即可下载。

2、ProxyEe Down

  1. ProxyEe Down 是使用本地HTTP代理服务器方式嗅探下载请求、支持所有操作系统和浏览器(IE9+)、支持多线程分段下载和断点下载。
  2. 简单的来说、就是安装这个插件后、让浏览器走这个软件建立的本地代理、访问百度网盘后、下载文件会自动转到 ProxyEe Down、然后就可以多线程下载了。

Github 项目：https://github.com/proxyee-down-org/proxyee-down
